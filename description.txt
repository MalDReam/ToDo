Это краткое описание проекта. Хотя насколько оно получится кратким - посмотрим в конце разработки.

Итак, приступим к анализу функциональных требований:
    1)Главное меню с указанием доступных команд;
    2)Создать заметку (Заголовок и описание);
    3)Удалить заметку (Выводится список всех заметок и нужно ввести номер заметки, которую мы хотим удалить);
    4)Просмотреть все заметки (Выводится список всех заметок с их описанием);
    5)Очистить список заметок (Удалить все заметки);
    6)Изменить заметку (Выбрать номер заметки и заново вписать все данные).

Хранение заметок производится с помощью локального файла.

После каждой команды должен выводиться список доступных команд из текущего местоположения пользователя.
После ввода команды экран должен самоочищаться.
